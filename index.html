Assigned variables:

You haven't assigned any variables yet, learn below how to do it.

Available methods:

Basic:

say
Returns a given argument
eg. say<"hello world">
clean
Cleans the output field
eg. clean

Maths:

add
Returns a sum of given arguments
eg. add<1 2>
subtract
Returns result of substraction
eg. subtract<1 2>
multiply
Returns result of multiplication
eg. multiply<1 2>
modulo
Returns result of modulo
eg. modulo<1 2>

Comparisons:

isGreaterThan
Tells if the first passed argument is greater than the second
eg. isGreaterThan<1 2>
isLessThan
Tells if the first passed argument is smaller than the second
eg. isLessThan<1 2>
isEqual
Tells if two arguements passed are equal
eg. isEqual<1 2>

Advanced functionalities:


assignVariable
Assigns a variable. This variable can be later used in other methods
eg. assignVariable<"x" 1>
if
If the first argument evaluates to true, it runs the second argument, otherwise it runs the third.
eg. if<isLessThan<2 1> say<"they are equal"> say<"they are not equal">>
while
It takes two expressions: the condition and the operation. The operation can consist of several operations. As long as the condition is met, while will run the operation. In order to run while properly you need to assign a variable first.
eg. assignVariable<"x" 1>
while<isLessThan<x 5> say<x> assignVariable<x add<x 1>>>
